# Frontend - Tenant App

React + TypeScript + Vite frontend for the Tenant App.

## Tech Stack

- React
- TypeScript
- Vite
- Material UI
- React Router
- Axios

## Prerequisites

- Node.js 18+
- npm

## Environment Variables

Create a `.env` file inside the `frontend` folder:

```env
VITE_API_URL=http://localhost:5000/api
```

## Install Dependencies

```bash
npm install
```

## Run in Development

```bash
npm run dev
```

App will run at the URL shown by Vite (usually `http://localhost:5173`).

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
